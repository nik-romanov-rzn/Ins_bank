# Ins_bank

# Сайт банка

Данный проект был выполнен в рамках учебной практической работы на 3 курсе колледжа РКЭ.

## О проекте
Это веб-сайт банка, разработанный с упором на пользовательский интерфейс и взаимодействие с пользователем.

Основная цель проекта — отработка навыков фронтенд-разработки: создание структуры страниц, работа с формами и реализация пользовательских сценариев.

Сайт реализован только для десктопной версии (без адаптации под мобильные устройства).

## Реализованный функционал

- Регистрация и вход пользователей (интерфейс и логика на стороне клиента)  
- Оформление заявки на банковскую карту  
- Форма обратной связи  
- Навигация по страницам сайта  
- Валидация форм и обработка пользовательского ввода  

## Технологии
- HTML  
- CSS  
- JavaScript  

## Особенности
- Проработанный пользовательский интерфейс  
- Чёткая структура страниц  
- Фокус на удобстве взаимодействия (UX)  
- Реализация типичных сценариев банковского сервиса  

## Примечание
Проект выполнен в учебных целях и демонстрирует навыки разработки интерфейсов.

В репозитории представлен статический вариант сайта для просмотра визуала и логики работы.
